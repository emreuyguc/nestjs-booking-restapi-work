## Nestjs Uibakery Booking rest api.

* Fastify Adapter
* Custom Configs
* Swagger
* Custom Validators
* Auto Validated Dtos
* TypeORM
* Dto | Class Mapping


## Configration
* ./core/data/configs/*.config.ts or use Cli params

## Installation

```bash
$ npm install
```

## Running the app

```bash
# With Config
$ DB_URL={$POSTGREURL} SERVER_PORT={$PORTNUMBER} npm run start

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
